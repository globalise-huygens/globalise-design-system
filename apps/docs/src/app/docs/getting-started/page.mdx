# Getting Started

The Globalise Design System is a React component library built with [React Aria Components](https://react-spectrum.adobe.com/react-aria/) and Tailwind CSS v4. It provides the UI building blocks for the Globalise project ecosystem.

> **Status:** This package is in active development (`v0.1.0`) and is **not yet published to npm**. It is currently only available within this monorepo. The instructions below describe the intended setup for when the package is released.

## Current development setup

If you're working within the monorepo, the docs app already consumes the design system via the pnpm workspace. No installation is needed â€” just run:

```bash
pnpm install
pnpm dev:docs
```

The example page at `/example` demonstrates all components in a realistic page layout.

## Future installation (once published)

The following instructions will apply once the package is published to npm.

### 1. Install the package

```bash
pnpm add @globalise/design-system
```

### 2. Import the styles

Add the design system stylesheet to your app's root layout or global CSS. This provides the design tokens (colours, fonts, spacing) as CSS custom properties:

```tsx
import "@globalise/design-system/styles.css";
```

Or in your CSS file:

```css
@import "@globalise/design-system/styles.css";
```

> **Note:** The stylesheet provides design tokens only. Your project must have its own Tailwind CSS v4 setup â€” the component classes are not pre-compiled.

### 3. Install fonts

The design system uses **Noto Sans** (UI/body text) and **Noto Serif** (headings/editorial). Load them via Google Fonts:

```html
<link
  href="https://fonts.googleapis.com/css2?family=Noto+Sans:wght@400;500;600;700&family=Noto+Serif:wght@400;500;700&display=swap"
  rel="stylesheet"
/>
```

### 4. Use components

Import and use components directly:

```tsx
import { Button, Typography, Container } from "@globalise/design-system";

export default function Page() {
  return (
    <Container>
      <Typography variant="h1">Hello, Globalise</Typography>
      <Button>Get started</Button>
    </Container>
  );
}
```

### Next.js setup

For Next.js projects, add `transpilePackages` to your `next.config.ts`:

```ts
const nextConfig = {
  transpilePackages: ["@globalise/design-system"],
};
```

## What's included

- **Layout primitives**: `Grid`, `Container`, `Section`, `Divider`
- **Cards**: `CardHero`, `CardFeatured`, `CardGlance`, `CardArticle`, `CardBase`
- **Navigation**: `Navbar`, `NavLink`, `NavLinks`, `NavSearchBar`
- **Content**: `Typography`, `Button`, `NewsletterSignup`, `SearchBar`, `ArticleRow`
- **Icons**: `IconAdd`, `IconArrowRight`, `IconChevronDown`, `IconClose`, `IconMenu`, `IconSearch`
- **Logos**: `Logomark`, `Logotype`, `LogoLockup`, `LogoNavbar`
- **Dev tools**: `GridGuide` (12-column overlay for alignment checking)
- **Design tokens** for brand colours, colour scales, typography, and spacing
- **Full TypeScript support** with exported types for every component
