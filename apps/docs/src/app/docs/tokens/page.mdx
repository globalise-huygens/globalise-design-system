## Tailwind Utilities

All tokens are registered with Tailwind CSS v4 via `@theme`, so you can use them as utility classes:

```css
/* Colors */
bg-brand-turquoise
text-neutral-500
border-vermilion-600

/* Fonts */
font-sans  /* Noto Sans */
font-serif /* Noto Serif */
```

## CSS Variables

You can also use the CSS custom properties directly:

```css
color: var(--brand-vermilion);
background-color: var(--turquoise-200);
font-family: var(--font-serif);
```
