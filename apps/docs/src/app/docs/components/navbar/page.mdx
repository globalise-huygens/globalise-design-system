# Navbar

A responsive navigation bar with logo, search bar, and navigation links. Includes a mobile hamburger menu that collapses on small screens.

## Preview

<ComponentPreview className="p-0! min-h-0!">
  <NavbarDemo />
</ComponentPreview>

## Usage

<CodeBlock code={`import { Navbar, NavLink, NavLinks, NavSearchBar } from "@globalise/design-system";
import { LogoNavbar } from "@globalise/design-system";

<Navbar logo={<LogoNavbar className="h-5 w-auto" />}>
  <NavSearchBar placeholder="Search the archive" />
  <NavLinks>
    <NavLink href="/about">About</NavLink>
    <NavLink href="/archive">Archive</NavLink>
    <NavLink href="/contact">Contact</NavLink>
  </NavLinks>
</Navbar>`} />

## Sub-components

The Navbar is composed of several sub-components:

- **`Navbar`**: the main navigation container
- **`NavSearchBar`**: a centered search input, hidden on mobile but visible in the mobile menu
- **`NavLinks`**: wraps navigation links, same visibility behaviour
- **`NavLink`**: individual navigation link
