# Button

A button component with multiple variants and sizes. Built on React Aria's `Button` primitive with full keyboard, focus, and accessibility support.

## Preview

<ComponentPreview>
  <div className="flex flex-wrap items-center gap-4">
    <Button>Default</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="link" className="text-white">
      Link CTA
    </Button>
    <Button variant="nav">Nav</Button>
  </div>
</ComponentPreview>

## Usage

<CodeBlock code={`import { Button } from "@globalise/design-system";

<Button variant="default">Get Started</Button>
<Button variant="outline">Learn More</Button>
<Button variant="link">Read article</Button>`} />

## Disabled State

All variants support a disabled state via `isDisabled`. Disabled buttons are dimmed and non-interactive.

<ComponentPreview>
  <div className="flex flex-wrap items-center gap-4">
    <Button isDisabled>Default</Button>
    <Button variant="outline" isDisabled>
      Outline
    </Button>
    <Button variant="link" isDisabled className="text-white">
      Link CTA
    </Button>
  </div>
</ComponentPreview>

<CodeBlock
  code={`<Button isDisabled>Disabled</Button>

## Link Variant as CTA

The `link` variant automatically includes a trailing arrow icon (`IconArrowRight`), making it the standard call-to-action link. Use `ButtonLink` with `variant="link"` for navigation links:

<ComponentPreview>
  <div className="flex flex-col gap-4">
    <ButtonLink variant="link" href="#" className="text-white">
      About the project
    </ButtonLink>
    <ButtonLink variant="link" href="#" className="text-brand-turquoise">
      Explore the archive
    </ButtonLink>
  </div>
</ComponentPreview>

<CodeBlock code={`import { ButtonLink } from "@globalise/design-system";

<ButtonLink variant="link" href="/about">About the project</ButtonLink>`} />

## Polymorphic Rendering

Use `as` to render the button styles on a different element (e.g. a span):

<CodeBlock code={`<Button as="span">Styled as span</Button>`} />

## ButtonLink

For link-styled buttons that navigate, use `ButtonLink`. It renders an accessible `<a>` element with React Aria's Link primitive:

<CodeBlock code={`import { ButtonLink } from "@globalise/design-system";

<ButtonLink href="/archive" variant="default">Browse Archive</ButtonLink>
<ButtonLink href="/about" variant="outline">Learn More</ButtonLink>`} />

## Interactive States

All button variants respond to hover, press, and focus states via React Aria.
