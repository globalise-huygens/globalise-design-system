# Button

A versatile button component with multiple visual variants and sizes. Supports polymorphic rendering via the `as` prop.

  </div>

## Usage

<CodeBlock code={`import { Button } from "@globalise/design-system";

<Button variant="default">Get Started</Button>
<Button variant="outline">Learn More</Button>
<Button variant="link">Read article →</Button>`} />

## Polymorphic Rendering

Use `as` to render the button styles on a different element (e.g. a span):

<CodeBlock code={`<Button as="span">Styled as span</Button>`} />

## ButtonLink

For link-styled buttons that navigate, use `ButtonLink`. It renders an accessible `<a>` element with React Aria's Link primitive:

<CodeBlock code={`import { ButtonLink } from "@globalise/design-system";

<ButtonLink href="/archive" variant="default">Browse Archive</ButtonLink>
<ButtonLink href="/about" variant="outline">Learn More</ButtonLink>`} />
