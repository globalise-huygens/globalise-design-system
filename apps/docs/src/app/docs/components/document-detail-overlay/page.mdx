# Document Detail Overlay

A near-full shell overlay for document scan and transcription workflows. It composes top and bottom bars, a left icon rail, a primary viewer pane, and a right transcript panel.

## Usage

<CodeBlock code={`import {
  DocumentDetailBody,
  DocumentDetailBottomBar,
  DocumentDetailIconRail,
  DocumentDetailOverlay,
  DocumentDetailSidePanel,
  DocumentDetailTopBar,
  DocumentDetailViewer,
  DocumentDetailViewerPane,
} from "@globalise/design-system";

<DocumentDetailOverlay isOpen={isOpen} onOpenChange={setIsOpen}>
  <DocumentDetailTopBar>{/* controls */}</DocumentDetailTopBar>

  <DocumentDetailBody>
    <DocumentDetailIconRail>{/* tool modes */}</DocumentDetailIconRail>

    <DocumentDetailViewer>
      <DocumentDetailViewerPane toolbar={<div>Viewer tools</div>}>
        {/* scan canvas */}
      </DocumentDetailViewerPane>
    </DocumentDetailViewer>

    <DocumentDetailSidePanel>{/* transcript */}</DocumentDetailSidePanel>

  </DocumentDetailBody>

  <DocumentDetailBottomBar>{/* status */}</DocumentDetailBottomBar>
</DocumentDetailOverlay>`} />

## Layout Notes

The default desktop placement uses a near-full 16-column shell span (`xl:col-start-2 xl:col-span-14`) so the viewer occupies most of the viewport while still leaving edge context.

Use `contentClassName` to override shell placement for alternate page or dual-view variants.
