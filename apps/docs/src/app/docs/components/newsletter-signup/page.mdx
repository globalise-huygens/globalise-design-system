# Newsletter Signup

A full-width newsletter subscription form with optional image, heading, description, email input, and privacy notice. Uses the brand turquoise background for the form area.

## Preview

<ComponentPreview background="light" className="p-0! min-h-0!">
  <div className="w-full">
    <NewsletterSignup
      heading="Stay up to date"
      description="Subscribe to the Globalise newsletter to receive updates on new collections, research, and events."
      inputPlaceholder="Your email address"
      submitLabel="Subscribe"
      privacyText="By subscribing you agree to our privacy policy."
    />
  </div>
</ComponentPreview>

## Usage

<CodeBlock code={`import { NewsletterSignup } from "@globalise/design-system";

<NewsletterSignup
  heading="Stay up to date"
  description="Subscribe to receive updates."
  inputPlaceholder="Your email address"
  submitLabel="Subscribe"
  privacyText={
    <>By subscribing you agree to our <a href="/privacy">privacy policy</a>.</>
  }
  image={<img src="/newsletter.jpg" alt="" className="object-cover w-full h-full" />}
  onSubmit={(e) => { e.preventDefault(); /* handle submit */ }}
/>`} />
