# Object Card Overlay

A grid-aware overlay shell for ObjectCard and future document scan or page viewer layouts. It wraps React Aria's `ModalOverlay` and `Modal`, then places content inside the centered 12-track slot of the 16-column shell grid using the shared shell gutter.

<ComponentPreview className="min-h-80! p-6! items-center! justify-center!">
  <ObjectCardOverlayDemo />
</ComponentPreview>

## Usage

<CodeBlock code={`import { ObjectCardOverlay } from "@globalise/design-system";

<ObjectCardOverlay isOpen={isOpen} onOpenChange={setIsOpen}>
  {children}
</ObjectCardOverlay>`} />

## Layout

This demo uses a neutral card shell rather than the Prins Eugenius content so you can judge modal coverage, backdrop depth, and the return/close state without the example content taking over the read.

Use `contentClassName` to shift, widen, or split the overlay content within the 16-column shell grid when you move beyond the default centered card layout.
