# Entity Highlight Menu

Dropdown menu pattern for entity highlight categories. It is driven by category rows with icons, counts, group selection, and optional child rows.

## Usage

<CodeBlock code={`<DocumentDetailPopoverSurface heading="Entity highlights" size="compact">
  {categories.map((category) => (
    <DocumentDetailCheckbox
      key={category.id}
      isSelected={selectedKeys.has(category.id)}
      onChange={(isSelected) => updateCategory(category.id, isSelected)}
    >
      {category.label}
    </DocumentDetailCheckbox>
  ))}
</DocumentDetailPopoverSurface>`} />
