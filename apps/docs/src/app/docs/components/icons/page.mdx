# Icons

A set of SVG icon components that inherit `currentColor` and accept all standard `<svg>` element props. Size them with `width` and `height` or Tailwind utilities.

## Usage

<CodeBlock code={`import { IconSearch, IconClose } from "@globalise/design-system";

<IconSearch className="w-5 h-5 text-white" />
<IconClose className="w-4 h-4 text-white/60" />`} />

## Sizing

Icons have no intrinsic size. Set `width`/`height` or use Tailwind classes to size them.

<ComponentPreview>
  <div className="flex items-end gap-6 text-brand-turquoise">
    <IconSearch className="w-4 h-4" />
    <IconSearch className="w-6 h-6" />
    <IconSearch className="w-8 h-8" />
    <IconSearch className="w-12 h-12" />
  </div>
</ComponentPreview>

## Colour

Icons use `currentColor` by default, so they inherit the text colour of their parent.

<ComponentPreview>
  <div className="flex gap-6">
    <IconAdd className="w-8 h-8 text-brand-turquoise" />
    <IconAdd className="w-8 h-8 text-brand-vermilion" />
    <IconAdd className="w-8 h-8 text-brand-mint" />
    <IconAdd className="w-8 h-8 text-white" />
  </div>
</ComponentPreview>
