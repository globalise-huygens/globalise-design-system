# Card Hero

An interactive card that reveals a branded color panel with label, title, and arrow on hover. Used in a hero section or featured content grids.

## Preview

<ComponentPreview className="min-h-75!">
  <CardHeroDemo />
</ComponentPreview>

## Usage

<CodeBlock code={`import { CardHero } from "@globalise/design-system";

<CardHero
  label="Archive"
  title="Explore the VOC\\nCorrespondence"
  hoverColor="turquoise"
  href="/archive"
>
  <img src="/hero.jpg" alt="" className="object-cover w-full h-full" />
</CardHero>`} />

## Hover Colors

The `hoverColor` prop sets the background on hover:

- `turquoise` (default)
- `vermilion`
- `mint`
- `parchment`
