# Reference Panel

A reusable reference list for object cards, document tables of contents, scan hits, and voyage references. The item layout uses the document-viewer scan card language: active left border, optional thumbnail, compact heading/action row, optional two-line snippet, metadata row, link action, and copy URI action.

## Usage

<CodeBlock code={`import { ReferencePanel } from "@globalise/design-system";

const references = [
  {
    id: "1764-0054-264",
    title: "1764 · 0054 · p.264",
    snippet: "in 't geheel p:r de prins Eugenius...",
    metadata: "NL-HaNA 1.04.02 · 10070",
    href: "#",
    uri: "https://example.com/reference/1764-0054-264",
  },
];

<ReferencePanel title="References (1)" items={references} />`} />
