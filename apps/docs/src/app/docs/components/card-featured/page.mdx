# Card Featured

An accordion-style card component that expands one item at a time. Each collapsed item shows a brand-colored strip; the expanded item reveals full content with optional background image.

## Preview

<ComponentPreview className="min-h-87.5! p-0!">
  <CardFeaturedDemo />
</ComponentPreview>

## Usage

<CodeBlock code={`import { CardFeatured } from "@globalise/design-system";

<CardFeatured
  items={[
    {
      color: "var(--brand-turquoise)",
      label: "Collection",
      title: "VOC Archives\\nExplored",
      cta: "View collection",
      href: "/collections/voc",
    },
    {
      color: "var(--brand-vermilion)",
      label: "Research",
      title: "Maritime Trade\\nRoutes",
      cta: "Read more",
      href: "/research/trade",
    },
  ]}
  defaultExpanded={0}
/>`} />
