# Divider

A horizontal rule component with the brand's semi-transparent white styling. `SectionDivider` wraps `Divider` inside a `Container` with vertical spacing.

    </Section>
    <Section background="light" spacing="default">
      <Container>
        <Divider color="dark" />
      </Container>
    </Section>
  </div>
</ComponentPreview>

## Usage

<CodeBlock code={`import { Divider, SectionDivider } from "@globalise/design-system";

{/* Simple divider */}

<Divider />

{/* Full-width section divider with container + spacing */}

<SectionDivider />`} />

## SectionDivider

`SectionDivider` combines a `Container` with a `Divider` and adds `my-s96` vertical margin. Use it between major page sections when you want the standard 8px-major rhythm spacing.
