# Segmented Buttons

Material 3-style React Aria toggle groups for compact viewer mode choices.

## Preview

<ComponentPreview>
  <DocumentDetailSegmentedToggleGroup
    aria-label="Viewer visibility"
    selectionMode="multiple"
    defaultSelectedKeys={["scan", "text"]}
  >
    <DocumentDetailSegmentedToggleItem id="scan" icon={<IconScan />}>
      Scan
    </DocumentDetailSegmentedToggleItem>
    <DocumentDetailSegmentedToggleItem id="text" icon={<IconTranscription />}>
      Text
    </DocumentDetailSegmentedToggleItem>
  </DocumentDetailSegmentedToggleGroup>
</ComponentPreview>

## Usage

<CodeBlock
  code={`<DocumentDetailSegmentedToggleGroup
  aria-label="Viewer visibility"
  selectionMode="multiple"
  selectedKeys={selectedKeys}
  onSelectionChange={setSelectedKeys}
>
  <DocumentDetailSegmentedToggleItem id="scan" icon={<IconScan />}>
    Scan
  </DocumentDetailSegmentedToggleItem>
  <DocumentDetailSegmentedToggleItem id="text" icon={<IconTranscription />}>
    Text
  </DocumentDetailSegmentedToggleItem>
</DocumentDetailSegmentedToggleGroup>`}
/>
