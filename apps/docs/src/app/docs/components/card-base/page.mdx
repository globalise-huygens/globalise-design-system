# Card Base

A composable, low‑level card primitive with colour variants. Use it to build custom card layouts with `CardBaseHeader`, `CardBaseTitle`, `CardBaseDescription`, `CardBaseContent`, and `CardBaseFooter`.

## Usage

<CodeBlock code={`import {
  CardBase,
  CardBaseHeader,
  CardBaseTitle,
  CardBaseDescription,
  CardBaseContent,
  CardBaseFooter,
} from "@globalise/design-system";

<CardBase variant="teal">
  <CardBaseHeader>
    <CardBaseTitle>Hello</CardBaseTitle>
    <CardBaseDescription>Subtitle</CardBaseDescription>
  </CardBaseHeader>
  <CardBaseContent>Body</CardBaseContent>
  <CardBaseFooter>Footer</CardBaseFooter>
</CardBase>`} />

### Sub‑components

`CardBaseHeader`, `CardBaseTitle`, `CardBaseDescription`, `CardBaseContent`, and `CardBaseFooter` accept standard `HTMLDivElement` props (`className`, `children`, etc.) and forward refs.
