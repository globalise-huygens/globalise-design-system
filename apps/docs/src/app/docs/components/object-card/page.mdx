# Object Card

A composable card built on React Aria Components for displaying entity detail overlays (Ship, Person, etc). All sub-components are pre-styled — consumers compose them without writing any CSS. On smaller screens, panels stack into one continuous scroll below the sticky header; on larger screens, panels sit side-by-side and scroll independently.
