# Card Glance

A branded info card with heading, subtitle, description, and call-to-action. Uses brand colors as background.

## Preview

<ComponentPreview>
  <CardGlanceDemo />
</ComponentPreview>

## Usage

<CodeBlock code={`import { CardGlance } from "@globalise/design-system";

<CardGlance
  heading="2020"
  subtitle="Project launch"
  description="GLOBALISE started unlocking VOC archives."
  cta="Read more"
  color="turquoise"
  href="/about"
/>`} />

## Colors

Available `color` variants: `turquoise` (default), `vermilion`, `mint`, `parchment`.
