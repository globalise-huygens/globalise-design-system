# Container

Places content in **columns 2 to 11** of the 12-column layout grid. Constrains to 1440px max-width, centers itself, and applies 1-column padding on each side at desktop. The outer columns stay free for full-bleed elements.

## Usage

<CodeBlock code={`import { Container } from "@globalise/design-system";

<Container>
  <h1>Page content here</h1>
  <p>Automatically inset by 1 column on each side at desktop.</p>
</Container>`} />

## Responsive padding

On smaller screens the grid padding scales down proportionally:

| Breakpoint     | Padding         | Equivalent    |
| -------------- | --------------- | ------------- |
| Default        | `16px`          | `px-4`        |
| `sm` (640px)   | `32px`          | `px-8`        |
| `lg` (1024px+) | `1/12 of width` | 1 grid column |

At 1440px viewport, `1/12 = 120px` padding on each side → 1200px content area (10 columns).
