# Container

Applies the shared page width and page margins for the standard content band inside the 16-column layout shell. Use it when you want the default centered content width without explicit column placement.

## Preview

<ComponentPreview>
  <div className="relative w-full">
    <Grid className="pointer-events-none absolute inset-0 w-full px-layout-page-margin-mobile lg:px-layout-page-margin">
      {Array.from({ length: 16 }).map((_, i) => (
        <div
          key={i}
          className={`h-full ${i < 2 || i > 13 ? "bg-neutral-600" : "bg-neutral-300"}`}
        />
      ))}
    </Grid>
    <Container className="relative w-full py-6">
      <div className="flex h-12 items-center justify-center bg-brand-turquoise text-sm font-mono text-brand-black">
        Standard content band
      </div>
    </Container>
  </div>
</ComponentPreview>

## Usage

<CodeBlock code={`import { Container } from "@globalise/design-system";

<Container>
  <h1>Page content here</h1>
  <p>Automatically inset by the shared page margins.</p>
</Container>`} />

## Responsive padding

Container uses the shared page-margin tokens:

| Breakpoint  | Padding |
| ----------- | ------- |
| Default     | `16px`  |
| `lg` and up | `32px`  |

At the page max width, that aligns Container with the same centered content band used by the 16-column grid.
