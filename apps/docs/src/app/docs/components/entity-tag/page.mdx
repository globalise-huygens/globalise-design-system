# Entity Tag

A linked inline entity treatment for named places, people, classified concepts, and thesaurus terms inside dense metadata or transcription text. Use EntityTag when the text itself is the entity; use EntityBadge when the UI needs a compact category marker.

## Usage

<CodeBlock code={`import { EntityTag } from "@globalise/design-system";

<EntityTag type="place" href="/entities/amsterdam">
  Amsterdam
</EntityTag>`} />
