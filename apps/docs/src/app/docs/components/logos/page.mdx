# Logos

Brand logo components rendered as inline SVGs. They inherit `currentColor` and accept standard `<svg>` element props.

## Variants

### Logomark

The Globalise symbol mark on its own.

<ComponentPreview>
  <Logomark className="h-20 text-brand-white" />
</ComponentPreview>

### Logotype

The word "Globalise" in the brand typeface.

<ComponentPreview>
  <Logotype className="h-8 text-brand-white" />
</ComponentPreview>

### LogoLockup

Symbol + word mark combined.

<ComponentPreview>
  <LogoLockup className="h-14 text-brand-white" />
</ComponentPreview>

### LogoNavbar

Compact horizontal logo sized for navigation bars.

<ComponentPreview>
  <LogoNavbar className="h-6 text-brand-white" />
</ComponentPreview>
