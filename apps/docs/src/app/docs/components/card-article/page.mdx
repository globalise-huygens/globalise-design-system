# Card Article

A content card for news and articles with an optional image, label, title, and "Read more" call-to-action. Renders as a link when `href` is provided, and does not include any hovering animation over the image/coloured background. Planned to be used in the content pages only.

## Preview

<ComponentPreview>
  <div className="max-w-sm text-brand-white">
    <CardArticle
      label="Article"
      title="VOC Archives Reveal New Maritime Trade Routes"
      image={<div className="w-full h-full bg-parchment-600" />}
    />
  </div>
</ComponentPreview>

## Usage

<CodeBlock code={`import { CardArticle } from "@globalise/design-system";

<CardArticle
  label="News"
  title="New Handwritten Text Recognition models released"
  href="/news/htr-models"
  image={<img src="/article-image.jpg" alt="" className="object-cover w-full h-full" />}
/>`} />
