# SearchBar

A search input field built with React Aria's `SearchField`. Supports multiple visual variants and sizes.

## Preview

<ComponentPreview>
  <div className="flex flex-col gap-4 w-full max-w-md">
    <SearchBar placeholder="Search the archive" />
  </div>
</ComponentPreview>

## Usage

<CodeBlock code={`import { SearchBar } from "@globalise/design-system";

<SearchBar placeholder="Search the archive" />`} />
